== WhoRun.It

A disributed, social running club for the web.

== Setup

Please edit the local_env.yml.sample with your particular environment variables. Application is using:
* GOOGLE_MAPS_KEY (DEV/TEST/PROD)
* GMAIL_USERNAME (DEV/TEST)
* GMAIL_PASSWORD (DEV/TEST)
* SENDGRID_USERNAME (PROD)
* SENDGRID_PASSWORD (PROD)
